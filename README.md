# LinkHub 🔗

> Encurtador de links com QR code e analytics de cliques em tempo real.

## Stack
Next.js • PWA • Redis • QR Code API

## Funcionalidades
- URLs de 6 caracteres
- QR code dinamico (1 clique)
- Analytics com fingerprint
- Expiracao programada
- PWA instalavel

## Metrica
QR code gerado em 1 clique — analytics em tempo real.
